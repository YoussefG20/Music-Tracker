## Description
This application is a Book Gallery, that allows users to efficiently search million's books that range from physical to digital copies within a single google search. The gallery involves a beautifully made interface that will showcase the picture of the book and information for the user to do further research if they desire. 

## Installation
Simply npm install and the according dependencies will be installed

## Launch Instructions
To run this application, follow these steps:
1. Install Node.js if you haven't already. You can download it from [https://nodejs.org/](https://nodejs.org/).

2. Open your terminal or command prompt and navigate to the project's directory.

3. To download the dependencies please type npm install in the terminal

4. Run the following command to start the application: node server.js

5. The application should now be running. You can access it by opening a web browser and navigating to [http://localhost:PORT]

## Testing Instructions

1. Ensure that the application is running as per the Launch Instructions.

2. You can then search any song you like and get 10 suggestions from a library of millions of books around the world.
